Step-by-step guide
1

Get Access

Smart Soho will share both sheets with you via email.

  • Accept both sharing invites — edit access to both sheets is required for Sync to work.
  • To add another team member, contact Smart Soho or share directly from Google Sheets.
2

Open the Booking Sheet

The only sheet you'll ever edit.

  • Open the Borough Yards Booking Sheet from the link shared with you.
  • You'll see columns for Space, Start date, End date, and Customer.
3

Edit Availability

Add, update, or remove rows as needed.

  • Add: fill in a new row — Space, Start date, End date, Customer.
  • Update: click any cell and type the new value.
  • Remove: select the row, right-click, delete.
  • Changes don't go live until you click Sync.
4

Click Sync

Pushes your changes live to the website.

  • Stay in the Borough Yards Booking Sheet — the Sync button is in the sheet itself.
  • Click it once to push your changes to the website.
  • Do this every time you make edits.
5

Done — Data is Live

The updated availability is now visible to enquiry form users.

  • Refresh the Hire a Space page — changes appear immediately.
  • No action needed on the Public Data Sheet; it's handled automatically.
